## 1.使用JMockit的基本格局

```java
   // 测试属性即要测试的那个类
    @Tested
    private HelloJMockit helloJMockit;
	
   // 依赖的接口
	@Injectable
    private XxxAPI xxxAPI;
 
    @Test
    public void testMethodName() {
        // 1。录制(Record)
        new Expectations() {
            {
              xxxAPI.apiMethod(); // mock的方法
              result = true;  // 设置改方法的返回值
            }
        };
        // 2.重放(Replay)
        helloJMockit.methodName(); // 测试的方法
        
            
        // 3.验证(Verification)
        new Verifications() {
            {
                xxxAPI.apiMethod();
                times = 1; // 验证调用的次数
            }
        };
    }
```
